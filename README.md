# Nyquist Rate demonstration


<img width="521" height="226" alt="image" src="https://github.com/user-attachments/assets/acb7988e-c5a9-474f-aa40-e65dcc7504be" />

> Nyquist rate is defined as the minimum sampling rate required to accurately capture an analog signal, which is at least twice the frequency of the highest oscillation present in the signal.

<hr>

This simple piece of code shows the effect of the sampling rate on a signal reconstruction and the phenomenon of aliasing for sample rates smaller the Nyquist Rate.
